Ruta: api/users/

Link https://playmusic.com.mx/api/users/

GET Method:

La API que se muestra a continuación, despliega una lista con todos los usuarios registrados. Como se puede apreciar, cada índice de la lista le corresponde a un usuario registrado único con su información personal que nos digitó. La contraseña está hasheada para respetar la privacidad del usuario.

  • {
           "user_id": 1,
           "password":"$2a$12$35nmzcnuNCHcK9p97e88Pe9E5MvtBd5I0F7iAL1NkA17S",
           "user_type": "User",
           "name": "Alex1",
           "user_name": "Aledqwdqwdx",
           "email": "gaelocote@gmail.com",
           "dob": "2002-09-23",
           "country": "Mexico"
    }

           "country": "Mexico"
    }


Ruta: api/user/ < int:pk>

Link https://playmusic.com.mx/api/user/1

GET Method:

Esta API muestra la información de un usuario en específico, según el número entero que se digita en el URL. En caso de digitar un número de un usuario inexistente, no mostrará nada.

  • {
           "user_id": 1,
           "password":"$2a$12$35nmzcnuNCHcK9p97e88Pe9E5MvtBd5I0F7iAL1NkA17S",
           "user_type": "User",
           "name": "Alex1",
           "user_name": "Aledqwdqwdx",
           "email": "gaelocote@gmail.com",
           "dob": "2002-09-23",
           "country": "Mexico"
    }

Ruta: api/sessions/

Link https://playmusic.com.mx/api/sessions/

GET Method:

La API referente a esta sección, muestra el momento en que un usuario inicio sesión en el sistema, así como el instante en el que cerró sesión. Para mayor precisión, se genera un Jason Web Token (jwt).

  • {
           "session_id": 1,
           "sign_in": "2022-04-08T05:50:45.786775Z",
           "sign_out": "2022-04-08T05:50:45.786976Z",
           "user_id": 2,
           "jwt": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiIxM jMiLCJzaWduX2luIjoiMjAyMi0wNC0wOCAwNTo0Nzo1Mi44MTE2OTIiLCJlbWFpbCI6IjEyM0AxMjMuMTIzIn0.2V9yKLt-0JXVGptKuKvIQzstrRFXvS6kT1SeWB0Fk-Q"
    }


Ruta: api/levels/

Link https://playmusic.com.mx/api/levels/

GET Method:

También hay una API referente a los niveles del videojuego, en ella, podemos encontrar el nombre de la canción que le corresponde a cada nivel, la duración en segundos y la dificultad del nivel.

  • {
           "level_id": 7,
           "song_name": "botswana-richard-bodgers-main-version-00-30-7812",
           "difficulty": "Easy",
           "duration": 30
    }


Ruta: api/levels/

Link https://playmusic.com.mx/api/levels/

POST Method:

En esta API agregamso un nuevo nivel y necesitamso estos datos:

  • {
           "song_name": "botswana-richard-bodgers-main-version-00-30-7812",
           "difficulty": "Easy",
           "duration": 30
    }


Ruta: api/level/ < int:pk>

Link https://playmusic.com.mx/api/level/7

GET Method:

           "song_name": "botswana-richard-bodgers-main-version-00-30-7812",
           "difficulty": "Easy",
           "duration": 30
    }

Link https://playmusic.com.mx/api/level/7

PUT Method:

En esta API, actualizamos la información específica a un nivel, dependiendo del número digitado en la URL. En caso de escribir un número inexistente como id de un nivel, no se despliega nada.

Ruta: api/attempts/

Link https://playmusic.com.mx/api/attempts/

GET Method:

Con la API presentada, podemos encontrar información correspondiente a los intentos realizados por los usuarios. La información que se despliega es el nivel jugado, el estatus del intento (victoria o derrota), el id del usuario en cuestión y la puntuación obtenida.

  • {
           "attempt_id": 2,
            "level_id": 7,
           "status": true,
           "user_id": 7,
           "score": 2400
    }

Link https://playmusic.com.mx/api/levels/

POST Method:

En esta API validamos los datos del usuario para iniciar sesión en el sistema:

  • {
           "user_name": "Alex1",
           "password": "12345678",
    }


Ruta: api/logout/

POST Method:

    }
